Week 05

Javascript Basics

Numbers: Floating point numbers, decimals & Integers Strings: Sequence of characters, text Booleans: Logical Data (True or False) Undefined: Data type of a variable that does not have a value yet Null: Also means non-existent

Note that for String variable, the value must be enclosed using either the double quotes or the single quotes. (It’s more common to use the double quotes).

'8' === 8 returns false Because they are not the same data type or value. '8' === '8' returns true Because they are the same data type and value.

'8' !== 8 returns true Because they are not the same data type or value. '8' !== '8' returns false Because they are the same data type and value.

'Hello' == 'Goodbye' returns false because they are not the same value 5 == '5' returns true Because they have the same value although their data types are different

'Hello' != 'Goodbye' returns true because they are not the same string 'Hello' != 'Hello' returns false Because they are the same string

DOM Tree

getElementById(); Uses the value of an element's id attribute querySelector(); Uses a CSS selector, and returns the first matching element. Classes or Ids can be used

getElementsByClassName(); Selects all elements that have a specific value for their class attribute getElementsByTagName(); Selects all elements that have the specified tag name querySelectorAll() Uses a CSS selector to select all matching elements

parentNode Selects the parent of the current node previousSibling/nextSibling Selects the previous or next sibling from the DOM tree firstChild/lastChild Selects the first or last child of the current element

If you are working with the element more than once, it is best you store it in a variable. var itemOne = getElementById("one");

ids, classes and css selectors can also be used.

Week 08

jQuery

Scripts should be placed before the closing /body tag

• Script is not blocking other things from downloading • The DOM has already loaded by the time the script is executed

With jQuery, you can ✔Select or find elements ✔Update their content / size / visibility ✔Simplify event handling

$("div") – selects all /p div elements $("li") – selects all li elements $("p") – selects all p elements We have to select by CSS to narrow down

$("#page") = select any with id of 'page' $(".hot") = select any with class of 'hot' $("li.cool") = select only li with class of 'cool' $("li.cool a") = select only a that are inside li with class of 'cool'

Use $(document).ready(); to load JS only when page is ready

AJAX

AJAX is a technique for loading data into part of a page without having to refresh the entire page. The data is often sent in a format called JavaScript Object Notation (or JSON) AJAX The ability to load new content into part of a page improves the user experience because the user does not have to wait for an entire page to load if only a portion of the page is being updated.

Live search (or autocomplete) commonly uses Ajax. Eg. Google search. When you type into the search bar on the home page, sometimes you will see results coming up before you have finished typing

Websites with user-generated content (such as Twitter and Flickr) may allow you to display your information (such as your latest tweets or photographs) This involves collecting data from their servers or tapping on APIs
